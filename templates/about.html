System Architecture

About SMART LENDER

SMART LENDER is an intelligent ML underwriting tool that determines borrower eligibility and computes credit risk scores using eight optimized classifiers. The modular pipeline guarantees reproducible predictions with zero data leakage.

Training Pipeline

  1. Missing Value Imputation — Numerical columns imputed with training median; categorical with mode.
  2. Outlier Management — Log transform log(x+1) applied to skewed income and loan amount features.
  3. Feature Encoding — Binary categoricals label-encoded; multi-class variables one-hot encoded.
  4. Standard Scaling — Features normalized before passing to classifiers.
  5. Cross-Validation — 8 models compared across accuracy, precision, recall, and F1. Best selected automatically.
